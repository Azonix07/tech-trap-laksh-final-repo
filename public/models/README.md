# 3D Models Directory

Place your 3D model files here with these exact names:

## Required Files (GLB/GLTF format):

1. **motherboard.glb** - Main motherboard model
2. **ram.glb** - RAM stick model
3. **ssd.glb** - SSD drive model
4. **gpu.glb** - Graphics card model

## Model Requirements:

- **Format**: .glb or .gltf (GLB preferred)
- **Size**: Keep under 5MB each for fast loading
- **Origin**: Center the model at (0, 0, 0)
- **Scale**: Reasonable proportions (will be adjusted in code)
- **Textures**: Embedded in GLB file

## Tips:

- Use Blender to export as .glb
- Compress textures before export
- Remove unnecessary vertices/faces
- Test models at: https://gltf-viewer.donmccurdy.com/

## Once you add the models:

The 3D scene will automatically load and display them in Room 2.
